- Tôi đang cần làm 1 website để hiển thị Volume giao dịch của các Token - Binance Alpha bằng sử dụng api của Binance. Sử dụng ReactJS, Tailwind CSS, Vite để build website.

*APIs*
    - API - Token list: Documents = "https://developers.binance.com/docs/alpha/market-data/rest-api/token-list" 
    - API - Ticker 24hr: Documents = "https://developers.binance.com/docs/alpha/market-data/rest-api/24hr-ticker-price-change"

*UI*
    - Sử dụng hoàn toàn tiếng Việt
    - UI Đơn giản, dễ sử dụng
    - UI Responsive 
    - UI chỉ sử dụng theme dark

*Features*
    - Sau khi lấy danh sách token về sẽ có rất nhiều token được trả về chỉ hiển thị 30 token có volume cao nhất và hãy sắp xếp từ volume cao nhất đến thấp nhất.
    - Có chức năng nhập tên để search token.
